# 简介
- 商品信息管理系统，学校数据库课程作业，第一次用Vue组件式开发。
- 前后端分离，前端单页面应用。
- 利用Vue进行组件式开发，Webpack打包
- 前端：[zhaosql-front](https://github.com/mumuzsl/zhaosql-front)
# 技术栈
- Spring Boot
- JPA
- Vue
- Element
